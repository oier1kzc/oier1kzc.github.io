Ex. Count Unlabeled Graphs

有色图再放送。

Statement

给定正整数 n,m,p (1≤m≤n≤30,108≤p≤109,p is a prime)n, m, p\ (1 \leq m \leq n\leq 30, 10 ^ 8 \leq p \leq 10 ^ 9, p \text{ is a prime})n,m,p (1≤m≤n≤30,108≤p≤109,p is a prime)。

考虑由如下过程生成一张图

  • 生成任意一张 nnn 个点的无标号简单无向图。
  • 对于图的每个顶点，填上 111 到 mmm 的数字之一。
  • 需要保证对于 111 到 mmm 的每个数字，都至少填在了一个顶点上。

求可以生成出的不同的图个数，对 ppp 取模。

两张图被认为是不同的，当且仅当存在一组标号，分别将两张图的顶点标为 1,2,⋯ ,n1, 2, \cdots, n1,2,⋯,n，满足

  • 对于任意 xxx 满足 1≤x≤n1 \leq x \leq n1≤x≤n，两张图上点 xxx 的标号相同。
  • 对于任意 x,yx, yx,y 满足 1≤x<y≤n1 \leq x < y \leq n1≤x<y≤n，边 (x,y)(x, y)(x,y) 在一张图上存在，当且仅当它也在另一张图上存在。

因为置换下不动，所以对于一个环，它上面的所有颜色都是相同的。
